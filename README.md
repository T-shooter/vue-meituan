
# About

此项目是 vue仿美团外卖项目，接口数据都是从服务器实时获取的真实数据，十一假期所写，不少细节待完善，见谅。



## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + sass + axios


## 项目运行


```
git clone https://github.com/T-shooter/vue-meituan.git  

cd vue-meituan  

npm install

npm run dev 


访问: http://localhost:8080

```
